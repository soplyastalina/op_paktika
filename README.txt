=== Обработка Изображений ===

Это десктопное приложение на Python для базовой обработки изображений. Программа позволяет загружать изображения, делать снимки с веб-камеры и выполнять различные операции по изменению изображений.

=== Функционал Приложения

= Базовые возможности:

  - Загрузка изображений: Поддержка форматов `.png` и `.jpg`.
  - Захват с веб-камеры: Создание фотоснимков с подключенной веб-камеры.
  - Отображение каналов: Просмотр красного, зеленого или синего цветового канала изображения.
  - Сброс изображения: Возврат к исходному (загруженному или захваченному) изображению.

= Индивидуальный функционал по варианту:

  - Обрезка изображения: Обрезка по заданным пользователем координатам (x, y, ширина, высота)
  - Вращение изображения: Поворот изображения на заданный пользователем угол.
  - Рисование прямоугольника: Нанесение синего прямоугольника на изображение по заданным координатам.

=== Стек Технологий

  Python: Версия 3.8
  Графический интерфейс: PyQt5
  Обработка изображений: OpenCV (opencv-python)

=== Установка и Запуск

Для запуска приложения выполните следующие шаги:

1.  Клонируйте репозиторий:

    |   git clone https://github.com/soplyastalina/op_paktika.git
    |   cd op_paktika

2.  Создайте и активируйте виртуальное окружение:

    Для Windows:
      |   python -m venv venv
      |   venv\Scripts\activate

    Для Linux/macOS:
      |   python3 -m venv venv
      |   source venv/bin/activate

3.  Установите необходимые зависимости:

    |   pip install -r requirements.txt

4.  Запустите приложение:

    |   python main.py

=== Файл `requirements.txt`

Файл `requirements.txt` содержит точные версии всех библиотек, необходимых для работы приложения, что обеспечивает воспроизводимость окружения.

=== Возможные проблемы и решения

  - Проблемы с веб-камерой:
    Если приложение не может подключиться к веб-камере, убедитесь, что она не занята другим приложением, драйверы обновлены, и приложению даны необходимые разрешения на доступ к камере в настройках конфиденциальности вашей операционной системы.

  - Ошибки отображения изображений после обработки:
    Если возникают ошибки при отображении после обрезки или других манипуляций, это может быть связано с тем, что данные изображения не являются непрерывными в памяти или имеют некорректный тип данных для `PyQt5`. В коде предусмотрены проверки и преобразования для устранения таких проблем (`np.ascontiguousarray`, `astype(np.uint8)`). Убедитесь, что вы используете последнюю версию кода.
